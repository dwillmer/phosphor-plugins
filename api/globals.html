registerPlugin

  • registerPlugin(name: string): IDisposable
    • Defined in index.ts:214

    Register a plugin and load its JSON specification.

    throws

    An error if the plugin name is already registered.

    Notes

    A plugin name is the same as the name of the package which contains the plugin specification. For a plugin named my-plugin, this will load the my-plugin/package.json file. The phosphor-plugin field in that file will be used to configure the plugin.

    Parameters

    • name: string

      The name of the plugin to register.

    Returns IDisposable

    A disposable which will unload the plugin.
