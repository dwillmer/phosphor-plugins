Interface IExtension

An object which represents a contribution to an extension point.

Notes

All properties of an extension are treated as read-only.

Extensions are automatically disposed by the framework. This occurs when the plugin which registered the extension is unloaded, or when a manually registered extension is unregistered. Before an extension is disposed, it is first removed from any matching extension point.

See also registerPlugin, registerExtension
